## 插件化探索demo
![image_1ajt0p4g1es4g75iu51m6s127a9.png-11.2kB][1]

HotFix是宿主程序，HotPlugin是插件程序，不要在意工程名，后期还会探索热修复。
将HotPlugin的程序打包成apk，然后发送到sdcard，运行hotfix即可测试。详细效果见demo。
